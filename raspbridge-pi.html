Kanisa Generator

  2. aRtay - receives input from the user for the Kinvow
  3. Kinvow - displays aRtay values in the form of their Artyles
  4. Texioty - command line for dealing with the whole generator

Renderer

Kanisa Renderers should be designed and built through the pet method system for the best compatibility, although this isn't a requirement.

When artyles are present, you can modify any of the options that are available through the renderer. When using the Pet Method system, the renderer should have enough coding to handle how a simple pet operates.

A basic view of the RenderAppTemplate can be found here.
