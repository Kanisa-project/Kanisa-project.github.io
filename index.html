I am designing this art project with all computer skill levels in mind. If all you know how to use is a web browser, navigate to "Renderer" or "Gaims". If you're a bit more adventurous and know how to unzip a file, check out the "Generator". I want any person to be able to use any portion of this project. What started as a simple paint program has turned into something very entertaining.

x
You can download the generator here. Just find the downloaded file and unzip it, this will create the working directory with 3 folders and an application. Double click the application to run and it'll open a window with 4 main sections:

  1. Texioty
  2. IDUTC
  3. aRtay
  4. Kinvow
Example Renderer

The renderer can render Masterpiece files, there is quite a bit that goes into it, but I decided to include everything that is needed to work properly. This will create art for right now, with major plans in the future.
