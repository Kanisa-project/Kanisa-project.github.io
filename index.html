The Gallery has some images that I've been able to create within a Generator.


I am designing this art project with all computer skill levels in mind. If all you know how to use is a web browser, navigate to Renderer or Gaims. If you're a bit more adventurous and know how to unzip a file, check out the Generator.

The main goal of this project is to create 2 types of files: an image like these and a text file.

You can download the generator here. After downloading, find the file and unzip it, this will create the working directory with 3 folders and the generator. Double click the application to run and it'll open a window with 4 main sections:

  1. Texioty - Accepts commands and responses for questions.
  2. IDUTC - Sets the beginning of a masterpiece file.
  3. aRtay - Primary options for creating images.
  4. Kinvow - Main display of images being created.

Each section can display or modify pieces of masterpiece file, always one or the other, rarely both.

A renderer is anything that will accept a masterpiece file and render some type of anything from it.here

A simple entry point for Kanisa is the Gaim renderer. No need to download anything, just check here. It already has some example masterpiece files to choose from a plethora of different simple 2D games.
